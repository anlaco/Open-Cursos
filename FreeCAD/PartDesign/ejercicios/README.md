# Ejercicios - PartDesign

Esta carpeta contiene ejercicios prácticos para el workbench PartDesign de FreeCAD.

## Estructura

Los ejercicios están organizados por nivel de dificultad:
- Básico: Modelado de piezas simples con operaciones básicas
- Intermedio: Piezas con múltiples características y transformaciones
- Avanzado: Diseños complejos con técnicas avanzadas de modelado

## Formato

Cada ejercicio incluye:
- Plano o descripción de la pieza a modelar
- Objetivos de aprendizaje específicos
- Pistas sobre qué operaciones utilizar
- Archivo de solución (cuando corresponda)

## Contribuciones

Los ejercicios se irán añadiendo progresivamente a medida que se desarrolle el curso.
