# Ejemplos - PartDesign

Esta carpeta contiene ejemplos demostrativos del workbench PartDesign de FreeCAD.

## Contenido

Los ejemplos incluyen:
- Piezas mecánicas comunes (ejes, bridas, soportes)
- Uso de diferentes operaciones de modelado
- Diseños paramétricos
- Mejores prácticas de modelado

## Archivos

Cada ejemplo incluye:
- Archivo FreeCAD (.FCStd) con el modelo completo
- Descripción del proceso de modelado
- Explicación de decisiones de diseño
- Capturas de pantalla o renders

## Uso

Estos ejemplos pueden usarse como:
- Material de referencia durante el aprendizaje
- Plantillas para proyectos similares
- Estudio de técnicas de modelado profesional
