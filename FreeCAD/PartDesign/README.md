# Workbench PartDesign

## Introducción

El workbench PartDesign de FreeCAD permite crear piezas 3D sólidas mediante un enfoque paramétrico basado en características. Utiliza bocetos creados en Sketcher como base para generar sólidos 3D.

## Objetivos del Módulo

Al completar este módulo, serás capaz de:
- Crear sólidos 3D a partir de bocetos 2D
- Aplicar operaciones de modelado como extrusiones y revoluciones
- Utilizar características para modificar geometría
- Crear diseños paramétricos que puedan modificarse fácilmente

## Contenido

### 1. Fundamentos de PartDesign
- Concepto de Body (cuerpo)
- Creación de sólidos base
- Árbol de características
- Diseño paramétrico

### 2. Operaciones de Adición
- Pad (extrusión)
- Revolution (revolución)
- Loft (transición)
- Sweep (barrido)

### 3. Operaciones de Sustracción
- Pocket (vaciado)
- Hole (agujero)
- Groove (ranura)

### 4. Operaciones de Transformación
- Mirror (simetría)
- Linear Pattern (patrón lineal)
- Polar Pattern (patrón polar)
- MultiTransform (transformación múltiple)

### 5. Operaciones Adicionales
- Fillet (redondeo)
- Chamfer (chaflán)
- Draft (ángulo de desmoldeo)
- Thickness (espesor)

### 6. Técnicas Avanzadas
- Planos de referencia
- Ejes de referencia
- Operaciones booleanas
- Modelado de piezas complejas
