# Workbench Sketcher

## Introducción

El workbench Sketcher de FreeCAD es la herramienta fundamental para crear bocetos 2D que servirán como base para diseños 3D más complejos. Los bocetos pueden incluir líneas, círculos, arcos y otras formas geométricas con restricciones que definen sus relaciones.

## Objetivos del Módulo

Al completar este módulo, serás capaz de:
- Crear bocetos 2D precisos utilizando las herramientas de dibujo
- Aplicar restricciones geométricas para definir relaciones entre elementos
- Utilizar restricciones dimensionales para controlar tamaños y distancias
- Crear bocetos completamente restringidos (fully constrained)

## Contenido

### 1. Herramientas de Dibujo Básicas
- Líneas y polilíneas
- Rectángulos y polígonos
- Círculos y arcos
- Elipses y otras curvas

### 2. Restricciones Geométricas
- Coincidencia de puntos
- Paralelismo y perpendicularidad
- Horizontalidad y verticalidad
- Tangencia
- Simetría

### 3. Restricciones Dimensionales
- Distancias
- Ángulos
- Radios y diámetros
- Longitudes

### 4. Técnicas Avanzadas
- Geometría de construcción
- Geometría externa
- Clonación de bocetos
- Validación de bocetos
