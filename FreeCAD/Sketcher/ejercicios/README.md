# Ejercicios - Sketcher

Esta carpeta contiene ejercicios prácticos para el workbench Sketcher de FreeCAD.

## Estructura

Los ejercicios están organizados por nivel de dificultad:
- Básico: Ejercicios introductorios para familiarizarse con las herramientas
- Intermedio: Ejercicios que combinan múltiples herramientas y restricciones
- Avanzado: Ejercicios complejos que requieren planificación y técnicas avanzadas

## Formato

Cada ejercicio incluye:
- Descripción del problema
- Objetivos de aprendizaje
- Pistas o sugerencias
- Archivo de solución (cuando corresponda)

## Contribuciones

Los ejercicios se irán añadiendo progresivamente a medida que se desarrolle el curso.
