# Ejemplos - Sketcher

Esta carpeta contiene ejemplos demostrativos del workbench Sketcher de FreeCAD.

## Contenido

Los ejemplos incluyen:
- Bocetos básicos con restricciones simples
- Uso de geometría de construcción
- Técnicas de bocetado eficientes
- Casos de uso comunes en diseño mecánico

## Archivos

Cada ejemplo incluye:
- Archivo FreeCAD (.FCStd) con el boceto completo
- Descripción de las técnicas utilizadas
- Capturas de pantalla o imágenes de referencia

## Uso

Estos ejemplos pueden usarse como:
- Material de referencia durante el aprendizaje
- Plantillas para proyectos propios
- Inspiración para resolver problemas similares
