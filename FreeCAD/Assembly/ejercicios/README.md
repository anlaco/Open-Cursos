# Ejercicios - Assembly

Esta carpeta contiene ejercicios prácticos para el workbench Assembly de FreeCAD.

## Estructura

Los ejercicios están organizados por nivel de dificultad:
- Básico: Ensamblajes simples con pocas piezas y restricciones básicas
- Intermedio: Ensamblajes con múltiples componentes y restricciones cinemáticas
- Avanzado: Ensamblajes complejos con sub-ensamblajes y movimientos

## Formato

Cada ejercicio incluye:
- Piezas individuales o instrucciones para crearlas
- Descripción del ensamblaje final deseado
- Objetivos de aprendizaje específicos
- Pistas sobre restricciones a utilizar
- Archivo de solución (cuando corresponda)

## Contribuciones

Los ejercicios se irán añadiendo progresivamente a medida que se desarrolle el curso.
