# Ejemplos - Assembly

Esta carpeta contiene ejemplos demostrativos del workbench Assembly de FreeCAD.

## Contenido

Los ejemplos incluyen:
- Ensamblajes mecánicos comunes (bisagras, mecanismos, articulaciones)
- Uso de diferentes tipos de restricciones
- Sub-ensamblajes y jerarquías
- Movimientos y simulaciones cinemáticas

## Archivos

Cada ejemplo incluye:
- Archivos FreeCAD (.FCStd) con las piezas individuales
- Archivo FreeCAD del ensamblaje completo
- Descripción del proceso de ensamblaje
- Explicación de las restricciones utilizadas
- Capturas de pantalla o animaciones

## Uso

Estos ejemplos pueden usarse como:
- Material de referencia durante el aprendizaje
- Plantillas para ensamblajes similares
- Estudio de técnicas de ensamblaje profesional
- Base para proyectos más complejos
