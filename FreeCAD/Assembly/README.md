# Workbench Assembly

## Introducción

El workbench Assembly de FreeCAD permite ensamblar múltiples piezas creadas en PartDesign, definir restricciones entre ellas y simular movimientos. Es fundamental para crear productos complejos que constan de múltiples componentes.

## Objetivos del Módulo

Al completar este módulo, serás capaz de:
- Importar y organizar múltiples piezas en un ensamblaje
- Aplicar restricciones para posicionar piezas correctamente
- Definir relaciones cinemáticas entre componentes
- Validar interferencias y colisiones
- Exportar ensamblajes completos
