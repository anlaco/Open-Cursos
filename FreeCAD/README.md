# Curso de FreeCAD

Este curso cubre el uso de FreeCAD, un software de diseño paramétrico 3D de código abierto.

## Descripción

FreeCAD es una herramienta poderosa para diseño 3D, especialmente útil para diseño mecánico, pero también puede usarse en otros campos como arquitectura y diseño de productos.

## Contenido del Curso

Este curso está organizado en tres módulos principales, cada uno enfocado en un workbench específico de FreeCAD:

### 1. [Sketcher](./Sketcher/)
El workbench Sketcher permite crear bocetos 2D con restricciones geométricas. Es la base para crear diseños más complejos en 3D.

**Temas cubiertos:**
- Herramientas de dibujo básicas
- Restricciones geométricas
- Restricciones dimensionales
- Técnicas de bocetado

### 2. [PartDesign](./PartDesign/)
El workbench PartDesign permite crear piezas 3D sólidas utilizando un enfoque de diseño basado en características.

**Temas cubiertos:**
- Creación de sólidos por extrusión
- Operaciones de revolución
- Paddings y pockets
- Operaciones booleanas
- Patrones y simetrías

### 3. [Assembly](./Assembly/)
El workbench Assembly permite ensamblar múltiples piezas y definir relaciones entre ellas.

**Temas cubiertos:**
- Importación de piezas
- Restricciones de ensamblaje
- Relaciones entre componentes
- Simulación de movimientos

## Prerrequisitos

- Instalación de FreeCAD (versión 0.20 o superior recomendada)
- Conocimientos básicos de geometría
- Comprensión de conceptos de diseño 3D (recomendado pero no obligatorio)
