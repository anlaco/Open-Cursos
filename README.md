# Open-Cursos

Este repositorio tiene el objetivo de agrupar el material para mis cursos open.

## Cursos Disponibles

### FreeCAD

Curso completo de FreeCAD cubriendo los siguientes workbenches:

- **Sketcher**: Herramientas de dibujo 2D y creación de bocetos
- **PartDesign**: Diseño de piezas 3D basado en características
- **Assembly**: Ensamblaje de piezas y componentes

Para más información sobre cada workbench, consulta la carpeta correspondiente en `/FreeCAD/`
